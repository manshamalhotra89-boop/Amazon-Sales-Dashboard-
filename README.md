# Amazon-Sales-Dashboard-
Data analysis project using Power BI to explore Amazon retail sales data, visualize revenue distribution, and identify business insights.

**Objective:**
The objective of this project is to analyze retail sales data and create an interactive dashboard to understand revenue distribution, product category performance, payment methods, and regional sales patterns.

**Tools Used**
Microsoft Power BI
Microsoft Excel

**Key Insights**
Revenue is distributed across multiple product categories, showing balanced sales performance.
Different payment methods are used by customers, indicating varied purchasing preferences.
Regional analysis helps identify areas generating higher revenue.

**Business Recommendation**
Focus marketing strategies on regions generating high revenue to increase sales further.
Promote popular product categories to maximize revenue potential.
Optimize payment options to improve customer purchase experience.
